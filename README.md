### Snabb start-guide:

Alternativ 1:

Alternativ 2:
```bash
# 1. Navigera till projektmappen
# 2. Skapa virtual environment
python3 -m venv venv
# 3. Aktivera virtual environment
source venv/bin/activate
# 4. Installera dependencies
pip install -r requirements.txt
# 5. Starta servern
python server.py
```
